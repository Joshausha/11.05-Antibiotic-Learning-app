## 🎯 Implementation Task: Design 1 - Floating Sidebar Dashboard

### Goal:
Transform the current layout to maximize visualization space with a collapsible sidebar.

### Target Architecture:
```
┌─────────────────────────────────────────────────┐
│ Header (40px) [☰ Menu] [Title] [Layout Select] │
├─────────────────────────────────────────────────┤
│                                        │        │
│                                        │ Sidebar│
│        Full-Screen                     │ (280px)│
│        Cytoscape                       │ Filters│
│        Network                         │ Legend │
│        Visualization                   │ (Scroll│
│        (100% when closed)              │  able) │
│                                        │        │
│                                        │        │
└─────────────────────────────────────────────────┘
    [Semi-transparent backdrop when open]
```

### Key Features:
- 🎯 100% visualization space when sidebar closed
- 🎯 280px collapsible sidebar with smooth slide animation
- 🎯 Semi-transparent backdrop overlay
- 🎯 Minimal 40px header with hamburger menu
- 🎯 Keyboard accessibility (ESC to close)
- 🎯 ARIA labels for screen readers
- 🎯 Mobile-responsive design

```javascript
const toggleSidebar = () => {
  setIsSidebarOpen(prev => !prev);
};

// Keyboard accessibility - ESC to close
useEffect(() => {
  const handleEscape = (e) => {
    if (e.key === 'Escape' && isSidebarOpen) {
      setIsSidebarOpen(false);
    }
  };
  document.addEventListener('keydown', handleEscape);
  return () => document.removeEventListener('keydown', handleEscape);
}, [isSidebarOpen]);
```

## 🔧 Troubleshooting

### Problem: Sidebar doesn't slide, just appears/disappears
**Solution:** Check Tailwind transition classes are present:
```javascript
className="... transition-transform duration-300 ease-in-out"
```

### Problem: Backdrop doesn't appear
**Solution:** Check conditional rendering:
```javascript
{isSidebarOpen && <div className="...backdrop..." />}
```

### Problem: Cytoscape doesn't fill screen
**Solution:** Check container classes:
```javascript
<div className="absolute inset-0 top-10">
```

### Problem: ESC key doesn't work
**Solution:** Check useEffect cleanup:
```javascript
return () => document.removeEventListener('keydown', handleEscape);
```

### Problem: Focus stuck in sidebar
**Solution:** Add focus trap or ensure tabindex is correct

### Problem: Mobile layout broken
**Solution:** Add responsive sidebar width:
```javascript
className="... w-80 sm:w-70"
```
