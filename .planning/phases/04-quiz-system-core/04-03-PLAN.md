Create QuizSession component to orchestrate the full quiz flow:

1. Define interface:
```typescript
interface QuizSessionProps {
  questions: QuizQuestion[];  // From medical.types.ts
  onComplete: (score: number, total: number) => void;
  onExit: () => void;
}
```

2. Implement state machine with phases:
   - 'active': Showing current question
   - 'explanation': Showing explanation after answer
   - 'complete': Showing results

3. State management:
```typescript
const [phase, setPhase] = useState<'active' | 'explanation' | 'complete'>('active');
const [currentIndex, setCurrentIndex] = useState(0);
const [score, setScore] = useState(0);
const [answers, setAnswers] = useState<Map<number, number>>(new Map());
```

4. Flow logic:
   - 'active': Show QuizQuestion, wait for answer selection
   - When answer selected: Record answer, check correctness, transition to 'explanation'
   - 'explanation': Show QuizExplanation, wait for "Next" click
   - When "Next" clicked: If more questions → 'active' with next question, else → 'complete'
   - 'complete': Show QuizResults

5. Key difference from QuizTab: NO auto-advance. User controls all navigation.

6. Pass appropriate props to child components:
   - QuizQuestion: question data, answer handlers, state
   - QuizExplanation: correctness, explanation text, onNext handler
   - QuizResults: score, handlers
