```typescript
interface QuizExplanationProps {
  isCorrect: boolean;
  explanation: string;
  correctAnswer: string;
  selectedAnswer: string;
  onNext: () => void;
  isLastQuestion?: boolean;
}
```

3. Implement the component with:
   - Distinct visual styling: green background (correct) vs amber/yellow background (incorrect)
   - Icon feedback: CheckCircle (correct) or XCircle (incorrect) from lucide-react
   - "Correct!" or "Incorrect — The answer is {correctAnswer}" header
   - Explanation section with:
     - "Explanation" heading in blue for visual hierarchy
     - Explanation text with good line-height and padding
   - Full-width button: "Next Question →" or "See Results" (if isLastQuestion)
   - User MUST click button to proceed (no auto-advance)

4. Follow codebase conventions:
   - Default export
   - React.FC<Props> type annotation
   - PascalCase component name
   - Lucide icons (CheckCircle, XCircle) - already used in QuizTab.tsx

Avoid: Auto-advance timers, complex animations, adding spaced repetition logic (Phase 5 scope).
