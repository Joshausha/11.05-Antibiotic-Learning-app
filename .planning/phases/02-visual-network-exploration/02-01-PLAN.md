  <action>
Create useD3ForceSimulation hook following React + D3 best practices from RESEARCH.md:

**Hook signature:**
```typescript
function useD3ForceSimulation(
  nodes: NetworkNode[],
  edges: NetworkEdge[],
  dimensions: { width: number; height: number }
)
```

**D3 force configuration:**
- d3.forceSimulation(nodes) with forces:
  - forceLink: distance=100, connects antibiotics to pathogens via coverage
  - forceManyBody: strength=-300 (repulsion prevents overlap)
  - forceCenter: pulls network to center of canvas
  - forceCollide: radius=30 (prevents node overlap)

**State management:**
- Store simulation in useRef (persist across renders)
- Update node/edge positions in React state
- **CRITICAL**: Throttle state updates to 50ms (not every tick) to avoid performance death
- Return { nodes, edges, simulation }

**Cleanup:**
- useEffect cleanup: simulation.stop()

**What to avoid and WHY:**
- Don't update state on every tick (60fps) → causes React re-render storm, performance death per RESEARCH.md
- Don't hand-roll physics → D3's force simulation handles collision, gravity, convergence
- Don't use d3.select for rendering → React controls DOM, D3 only for calculations
