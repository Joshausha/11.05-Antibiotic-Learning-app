Add connected edge highlighting when hovering a node:

1. In the node mouseenter handler (around line 425-447), add logic to highlight connected edges:
   - Find all edges where source or target matches hovered node
   - Use D3 selection to increase stroke-width and opacity of connected edges
   - Store original values so they can be restored

2. In the node mouseleave handler (around line 433-447), restore edge styles:

Implementation pattern:
```javascript
.on('mouseenter', (event, d) => {
  setHoveredNode(d);
  showNodeTooltip(event, d, layout, filteredEdges);

  // Highlight node
  d3.select(event.currentTarget)
    .transition()
    .duration(100)
    .attr('r', (current) => current + 3);

  // Highlight connected edges
  g.selectAll('.network-link')
    .attr('opacity', (e) => {
      const sourceId = (e.source || e.sourceId);
      const targetId = (e.target || e.targetId);
      return (sourceId === d.id || targetId === d.id) ? 0.9 : 0.15;
    })
    .attr('stroke-width', (e) => {
      const sourceId = (e.source || e.sourceId);
      const targetId = (e.target || e.targetId);
      const similarity = (e.similarity || e.strength || 0.5);
      return (sourceId === d.id || targetId === d.id)
        ? 3 + similarity * 4  // Boosted width
        : 1 + similarity * 2; // Reduced width
    });
})
```

Note: The `g` variable needs to be accessible in these handlers. Since D3 rendering is in a useEffect, you may need to store a ref to the group element or re-select it.
