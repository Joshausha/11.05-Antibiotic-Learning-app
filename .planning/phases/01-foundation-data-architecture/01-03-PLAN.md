**Medical Validation Requirements:**
- Northwestern 8-segment coverage ratings must be 0, 1, or 2 (no other values)
- Gram stain classifications must be 'positive' | 'negative' | 'variable' | 'not_applicable'
- All quiz questions must have medicalAccuracyVerified: true
- Antibiotic-pathogen relationships must be bidirectional (if antibiotic covers pathogen, pathogen lists antibiotic)
- Clinical guidelines must reference current medical standards

**Project Philosophy:**
- This is a learning tool, not a clinical decision support system
- Medical accuracy is non-negotiable for educational credibility
- Automated validation catches errors before they reach users
- Test coverage prioritizes medical accuracy over code coverage percentage
